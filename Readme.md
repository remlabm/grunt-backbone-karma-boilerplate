## Grunt Backbone Karma Boilerplate

Includes:
RequireJS
Backbone
CoffeeScript
Bootstrap / SaSS
Grunt
Mocha / Chai
Karma w/ PhantomJS

PHPStorm Config:
- install Karma plugin
- Add Runner for Karma
![PHPStorm Config](/phpstorm-config.png?raw=true)

PHPStorm Terminal:
```
$ npm install && bower install
$ grunt serve
```

Run Tests :)
